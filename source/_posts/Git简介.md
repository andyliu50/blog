# 什么是Git #

[Git](https://git-scm.com/)是分布式版本管理系统。作者是[Linux Torvalds](https://en.wikipedia.org/wiki/Linus_Torvalds)。开发者可以将Git程序安装在本地计算机上，并通过Git管理自己的代码。

# 什么是GitHub #

[GitHub](https://github.com/)是在线软件源代码托管服务，利用Git进行版本控制。
