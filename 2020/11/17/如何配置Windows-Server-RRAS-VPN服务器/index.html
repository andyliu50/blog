如何配置Windows Server RRAS VPN服务器

发表于 2020-11-17 | 更新于 2020-11-18

按照以下步骤，可以在Windows Server 2016上配置PPTP VPN服务器。

安装RRAS服务

  1. 打开Server Manager，选择Add roles and features。

  2. 点击Next。

  1. 点击Next。

  2. 点击Next。

  3. 选择Remote Access，点击Next。

  4. 点击Next。

  5. 点击Next。

  6. 选择DirectAccess and VPN(RAS)和Routing，点击Next。

  7. 点击Next。

  8. 点击Next。

  9. 点击Next。

  10. 点击Close。

初始配置

  1. 在Server Manager中，打开RRAS配置向导。

  2. 选择Deploy VPN Only。

  3. 右键选择WIN2016(local)，选择Configure and Enable Routing and Remote Access。

  4. 打开配置向导，点击Next。

  5. 选择Custom Configuration，点击Next。

  6. 选择VPN access, NAT, LAN routing，点击Next。

  7. 点击Finish。

配置认证方式

  1. 右键选择WIN2016(local)，选择Properties。

  2. 选择Security，选择Authentication Methods…。

  3. 选择MS-CHAP v2和CHAP两种认证方式。

配置IP地址池

如果内网没有DHCP服务器为VPN客户端分配IP地址，可以通过以下方式在RRAS服务器上配置IP地址池：

  1. 右键选择WIN2016(local)，选择Properties。

  2. 选择Security，选择IPv4。

  3. 选择Static address pool, 点击Add。

配置用户账号

  1. 打开计算机管理器
  2. 打开用户属性
  3. 选择Dial-in, 在Network Access Permission，选择Allow access。

客户端连接

  1. 打开Settings - Network&Internet。

  2. 选择VPN，点击**Add a VPN connection。

  3. 选择VPN类型为PPTP，并输入VPN服务器的IP地址或者名称，然后保存。

  4. 点击连接VPN。

  5. 在CMD中，输入ipconfig命令可以看到已经获取到VPN服务器分配的IP地址。

  6. 在CMD中，用tracert追踪到VPN服务器内网口IP地址(172.16.1.20)，以及Hyper-V主机的IP地址(172.16.1.1)，都会先经过172.16.1.100。该地址为为RRAS服务器虚拟网口(Internal)的IP地址。

  7. 在RRAS服务器上，打开CMD窗口，并输入ipconfig，可以看到PPP adapter RAS (Dial In) Interface的IP地址为172.16.1.100。
